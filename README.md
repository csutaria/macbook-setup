# macbook-setup
Tools and configuration files to set up a MacBook using Ansible

## Using macbook-setup
- Clone this repo
- Log in to the Mac App Store
- Run bootstrap.bash if needed, to install Homebrew and Ansible locally, and run the playbook
- Alternatively, if Ansible is already available, run `ansible-galaxy install -r requirements.yml` and then run playbook directly with ./playbook.yml
